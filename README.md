# FPS-Prototype

1. Controls

WASD 		- Walk
Space 		- Jump
C 		- Crouch
Esc		- Open Pause Menu
Mouse wheel 	- Swap weapon 
LeftMouseButton - Shoot


2. Game Scene

Yellow cubes are targets. HP = 50.

Common weapon attack power = 10;
Explosive weapon attack power = 20;
Explosive bullets push tasgets away.
Explosive weapon damage calculations includes distance to the target.
